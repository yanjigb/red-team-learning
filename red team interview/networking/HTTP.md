**(HyperText Transfer Protocol)**

Là cách trình duyệt sử dụng để gửi yêu cầu nhận traffic truy cập web

is what's used whenever you view a website, developed by Tim Berners-Lee and his team between 1989-1991

 HTTP is the set of rules used for communicating with web servers for the transmitting of webpage data, whether that is HTML, Images, Videos, etc.
