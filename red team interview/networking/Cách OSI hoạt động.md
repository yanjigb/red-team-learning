Layer 7 (App): Your browser builds an HTTP GET request
Layer 6 (Presentation): TLS encryption is negotiated (HTTPS)
Layer 5 (Session): A session is established with Google's server
Layer 4 (Transport): TCP segments the data, assigns Port 443
Layer 3 (Network): IP addresses are added — your IP + Google's IP
Layer 2 (Data Link): MAC addresses added for local delivery
Layer 1 (Physical): Bits travel as electrical/wireless signals
