File transfer protocol

Used for interactive file transfer between systems.

Là một giao thức client - server được sử dụng để truyền file qua mạng

FTP sử dụng TCP/IP

Sử dụng 2 kênh để trao đổi
1. Command channel: client device gửi lệnh đến server như yêu cầu file, hoặc dir
2. Data channel: để truyền file
