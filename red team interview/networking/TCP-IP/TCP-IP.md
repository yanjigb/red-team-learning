TCP/IP là hệ thống giúp dữ liệu đi từ máy A đến máy B bằng cách:
Tìm đường → Tìm đúng máy → Tìm đúng ứng dụng → Đảm bảo dữ liệu tới nơi.

### Gồm 4 tầng

```
Application
Transport
Internet
Network Access
```
